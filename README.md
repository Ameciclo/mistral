# Mistral

## Description
Mistral is a Node.js-based ETL (Extract, Transform, Load) project designed to streamline data operations related to the Observat√≥rio de Sinistros. It uses **pnpm** as the package manager and TypeScript for type-safe code.

## Prerequisites
1. **mise**
   - Install mise by following the instructions from the [official mise repository](https://mise.jdx.dev/).

## Setting Up the Environment
1. Install the required tools:
   ```
   mise install
   ```
2. Install project dependencies:
   ```
   pnpm install
   ```

## Usage
### Preprocessing Data
```
pnpm preprocess
```
